Trezor Bridge® | Introducing The New Trezor App

Trezor Bridge is a software that enables seamless communication between your Trezor device and compatible browsers.

Designed to operate in the background, Trezor Bridge requires no direct user interaction. Once installed, it runs silently, ensuring your Trezor device is always connected:

  • Windows: Open Task Manager and search for trezord
  • MacOS: Open Activity Monitor and search for trezord
  • Linux: Open System Monitor and search for trezord-go

How to Update Trezor Bridge

To update to the latest version of Trezor Bridge, visit this link and click on Download latest Bridge.

Download Trezor Bridge

This tool facilitates the connection between your Trezor device and your internet browser.

Current version: Trezor Bridge 2.0.27

For more details, visit the Github repository.

When using the desktop version of Trezor Suite, it will automatically connect to Trezor Bridge. If it fails, Trezor Suite will run its own instance of Trezor Bridge.

Checking Trezor Bridge Status

To check the status of Trezor Bridge, visit the Trezor Bridge status page.

Uninstalling Trezor Bridge

If you need to uninstall Trezor Bridge, follow the instructions for your operating system:

MacOS

  1. Open Finder and navigate to Applications > Utilities > TREZOR Bridge
  2. Double-click uninstall.pkg
  3. Follow the on-screen instructions provided by the "TREZOR Bridge Installer" package

Windows

  1. Open File Explorer and navigate to Program Files > TREZOR Bridge
  2. Double-click uninstall.exe
  3. Trezor Bridge will be uninstalled automatically

Linux

To remove Trezor Bridge, use the following command:

$ sudo apt remove trezor-bridge

To verify that Trezor Bridge has been uninstalled, run this command, which should return no results:

$ sudo apt list --installed | grep trezor-bridge
